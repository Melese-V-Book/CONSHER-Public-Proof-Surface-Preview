# Admissibility Map

The admissible posture is limited to public-safe repository-package preparation and public-exposure candidate review.

Admitted:

- repository structure
- non-claims visibility
- sanitized proof-surface description
- source-to-status route trace
- boundary-behavior table
- evidence-present and evidence-missing matrix
- refusal and downgrade table
- closed-with-limits interpretation
- author-domain boundary
- demo-only notice
- stakeholder and expert review invitation
- public-exposure candidate manifest

Not admitted:

- public exposure
- public release
- GitHub publication
- LinkedIn announcement
- deployment
- implementation
- operational use
- public advisory use
- visitor guidance
- heritage-site evaluation
- ritual interpretation
- public-safety instruction
- certification readiness
- production substrate
